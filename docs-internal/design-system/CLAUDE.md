# Project instructions

- Use properly readable fonts in all designs. Decorative/display theme fonts (UnifrakturCook, Pirata One, Major Mono Display, IM Fell English SC, etc.) may appear only as small specimens when documenting a theme — never on UI copy, buttons, labels, headings, or body text. UI copy always uses the readable stack: Geist for words, JetBrains Mono for machine voice, or each theme's readable body font.
- Avoid hard-to-read styles generally: no extreme letter-spacing on long text, no low-contrast text on tinted fills.
